Escalas

As escalas são a base para criar melodias, solos e compreender a estrutura das músicas. Neste módulo, vamos explorar as escalas mais importantes para guitarristas iniciantes. Vamos aprender como elas são formadas, como praticá-las e como utilizá-las na sua jornada musical.

Introdução às Escalas

O que são escalas? Uma escala é uma sequência de notas tocadas em ordem ascendente ou descendente.
Por que aprender escalas? Elas ajudam a desenvolver habilidades técnicas, improvisação, composição e compreensão musical.
Notas da escala: Cada escala é composta por um conjunto específico de notas. A escala maior, por exemplo, tem sete notas.

Escalas Maiores

A fórmula para a escala maior é: tom, tom, semitom, tom, tom, tom, semitom.

Dó Maior (C)
Notas: C, D, E, F, G, A, B, C

Dó Sustenido Maior (C#) / Ré Bemol Maior (Db)
Notas: C#, D#, E#, F#, G#, A#, B#, C#
Notas: Db, Eb, F, Gb, Ab, Bb, C, Db

Ré Maior (D)
Notas: D, E, F#, G, A, B, C#, D

Ré Sustenido Maior (D#) / Mi Bemol Maior (Eb)
Notas: D#, E#, F##, G#, A#, B#, C##, D#
Notas: Eb, F, G, Ab, Bb, C, D, Eb

Mi Maior (E)
Notas: E, F#, G#, A, B, C#, D#, E

Fá Maior (F)
Notas: F, G, A, Bb, C, D, E, F

Fá Sustenido Maior (F#) / Sol Bemol Maior (Gb)
Notas: F#, G#, A#, B, C#, D#, E#, F#
Notas: Gb, Ab, Bb, Cb, Db, Eb, F, Gb

Sol Maior (G)
Notas: G, A, B, C, D, E, F#, G

Sol Sustenido Maior (G#) / Lá Bemol Maior (Ab)
Notas: G#, A#, B#, C#, D#, E#, F##, G#
Notas: Ab, Bb, C, Db, Eb, F, G, Ab

Lá Maior (A)
Notas: A, B, C#, D, E, F#, G#, A

Lá Sustenido Maior (A#) / Si Bemol Maior (Bb)
Notas: A#, B#, C##, D#, E#, F##, G##, A#
Notas: Bb, C, D, Eb, F, G, A, Bb

Si Maior (B) / Dó Bemol Maior (Cb)
Notas: B, C#, D#, E, F#, G#, A#, B
Notas: Cb, Db, Eb, Fb, Gb, Ab, Bb, Cb

Escalas Menores Naturais

A fórmula para a escala maior é: tom, semitom, tom, tom, semitom, tom, tom.
